# Oracle--hotel-system-management

Ce projet met en place une base de données Oracle destinée à gérer l’activité d’un hôtel.
Il s’appuie sur l’architecture Multitenant (CDB/PDB) et utilise des tablespaces dédiés pour stocker les données et les index.
La base permet de gérer :
	• Les clients
	• Les hôtels
	• Les chambres
	• Les réservations
	• Les paiements
Des utilisateurs sont créés avec différents niveaux de privilèges (manager, réception, administrateur) afin de séparer les rôles métiers et administratifs.
